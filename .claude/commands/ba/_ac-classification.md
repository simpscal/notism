# AC Classification

When modifying acceptance criteria, classify each AC as:

| Type | Description |
|------|-------------|
| **Added** | New behaviour not covered by any existing AC |
| **Removed** | Existing behaviour that no longer applies (state original AC text) |
| **Modified** | Existing behaviour that is changing (state old -> new) |

## Output Format

After amendment, output:
```
Story #<N> amended.
ACs: Added <N> - Removed <N> - Modified <N>
```
