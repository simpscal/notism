# Issue User Story Template

User story issue body posted to GitHub. Used by `/ba` (standard mode S4, requirement-change mode RC4).

Read `.claude/templates/acceptance-criteria.md`.

```
## User Story
<As a ... I want ... so that ...>

<Acceptance Criteria + Notes from template>

---
Part of <link_to(requirement_issue_number)>
```

**User Story:** format — `As a <type of user>, I want <to perform some action> so that <I can achieve some goal/benefit>`

**Part of:** use `link_to(requirement_issue_number)` from the tracker adapter.
