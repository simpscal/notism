# Issue Requirement Template

Requirement issue body posted to GitHub. Used by `/po`.

```
## Summary
<2–3 sentence summary of what is needed and why>

## Goals
- <Goal 1>
- <Goal 2>

## Out of Scope
<Anything explicitly excluded, or "Not specified">
```

**Summary:** 2–3 sentences — what is needed and why.

**Goals:** bulleted list of explicit outcomes the requirement targets.

**Out of Scope:** anything explicitly excluded, or write "Not specified".
