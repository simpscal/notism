# Issue Bug Report Template

Bug report issue body posted to GitHub. Used by `/bug-report`.

```
## Bug Report

### Description
<concise summary of the defective behaviour>

### Steps to Reproduce
1. <step>
2. <step>
...

### Expected Behaviour
<what should happen>

### Actual Behaviour
<what actually happens>

### Severity
<critical | high | medium | low>
```

**Description:** concise summary of the defective behaviour.

**Steps to Reproduce:** numbered list of exact steps to trigger the bug.

**Expected Behaviour:** what should happen.

**Actual Behaviour:** what actually happens.

**Severity:** one of `critical | high | medium | low`.
